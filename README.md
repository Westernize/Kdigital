# kdigital 학원 학습 및 프로젝트 경험 정리

이 저장소는 kdigital 학원에서 7개월간(2025.07.01 ~ 2026.02.01) 체계적으로 학습한 내용을 정리한 자료입니다.  
실무 중심의 프로젝트와 다양한 실습을 통해 Java, JSP/Servlet, Oracle SQL 등 웹 개발 전반의 역량을 강화하였습니다.

## 폴더 구조

- `java/` : Java 기초 및 객체지향 프로그래밍 실습
- `jsp/` : JSP/Servlet 기반 웹 애플리케이션 개발 및 프로젝트
- `Oracle/` : Oracle SQL 실습 및 데이터베이스 연동 예제

## 주요 학습 및 프로젝트 경험

### 1. Java 기반 백엔드 개발
- 객체지향 설계, 예외 처리, 컬렉션 활용 등 실무에 필요한 Java 프로그래밍 역량 습득
- 파일 입출력, 네트워크 프로그래밍 등 확장성 있는 코드 작성 경험

### 2. 웹 애플리케이션 개발 (JSP & Servlet)
- MVC 패턴 기반의 웹 서비스 설계 및 구현
- 회원관리, 쇼핑몰 등 실전 프로젝트 수행
- 파일 업로드/다운로드, 세션/쿠키 관리, JSTL/EL 활용 등 다양한 기능 구현

### 3. 데이터베이스 설계 및 연동
- Oracle DB를 활용한 데이터 모델링 및 SQL 쿼리 작성
- DDL/DML/DCL, JOIN, 서브쿼리, 함수 등 실무 중심의 SQL 실습
- JDBC를 통한 Java-DB 연동 및 트랜잭션 처리

### 4. 실무 프로젝트 경험
- 요구사항 분석부터 설계, 구현, 테스트, 배포까지 전 과정 경험
- 협업 및 버전 관리(Git) 실습
- 코드 리뷰 및 문서화 경험

## 활용 및 기대 효과

- 실무에서 바로 적용 가능한 웹 개발 역량 보유
- 프로젝트 중심의 학습으로 문제 해결 능력 및 협업 경험 강화
- 다양한 예제와 프로젝트를 통한 포트폴리오 구축
