# kdigital 학원 학습 내용 정리

이 저장소는 kdigital 학원에서 학습한 내용을 정리한 자료입니다.  
각 챕터별 실습 코드, 프로젝트, SQL 예제, JSP/Servlet 실습 등 다양한 예제가 포함되어 있습니다.

## 폴더 구조

- `java/` : Java 기초 및 실습 예제
- `jsp/` : JSP/Servlet 실습 및 프로젝트
- `Oracle/` : Oracle SQL 실습 및 예제

## 주요 학습 내용

### 1. Java 기초 및 객체지향
- 변수, 자료형, 제어문, 배열
- 클래스와 객체, 상속, 인터페이스
- 예외 처리, 컬렉션 프레임워크

### 2. JSP & Servlet
- JSP 기본 문법, 내장 객체 활용
- Servlet 구조 및 동작 원리
- MVC 패턴 실습
- 파일 업로드/다운로드 구현
- JSTL, EL, 커스텀 태그 활용

### 3. 데이터베이스 & SQL
- Oracle DB 기본 사용법
- DDL, DML, DCL 명령어 실습
- JOIN, 서브쿼리, 함수 활용
- JDBC를 통한 DB 연동

### 4. 프로젝트 실습
- 쇼핑몰, 회원관리, 파일업로드 등 미니 프로젝트
- 실전 환경에서의 배포 및 테스트

## 사용 방법

각 폴더별로 README 또는 주석을 참고하여 실습 코드를 실행할 수 있습니다.


학습 내용을 지속적으로 업데이트할 예정입니다.
